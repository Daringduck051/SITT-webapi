## Support Interaction Theme Tally (SITT)
SITT is a program designed to help support officers tally interactions and provide them an easy way to report to whoever they need to report it to.

## Description
This project is designed for a support agent to be able to tally certain interactions that occur during their shift. The program allows a support agent to increase a tally based on the interaction, email a spreadsheet to someone else, create custom themes in case an interaction doesn't fit with current themes. After the email is sent, tallies/custom themes are locked until "New Shift" is clicked. Email is sent with a list of theme names and tallies as well as the time and date the shift ended.

## Usage
Internet Connection and a modern web browser. This project should run on any modern web browser without any problem. Created using .NET 10, SQLite, HTML/CSS/JavaScript all in Visual Studio Code. For a perfect replication, download all files into a main folder then open that folder in Visual Studio Code. Then run the following code in the terminal:

```bash
dotnet build
dotnet run
```
